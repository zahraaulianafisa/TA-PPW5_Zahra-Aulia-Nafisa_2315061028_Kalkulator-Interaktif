# TA-PPW5_Zahra-Aulia-Nafisa_2315061028_Kalkulator-Interaktif
<h1>Kalkulator Interaktif dengan Riwayat Perhitungan</h1>
Proyek ini adalah aplikasi Kalkulator Interaktif berbasis web yang dibuat menggunakan HTML + CSS + JavaScript.  
Aplikasi ini dilengkapi fitur riwayat perhitungan sehingga pengguna dapat melihat kembali hasil perhitungan sebelumnya.

Kalkulator ini dapat melakukan fungsi-fungsi berikut:
<li>Operasi Dasar: Penjumlahan, pengurangan, perkalian, pembagian, dan sisa hasil bagi
<li>Fungsi Trigonometri: sin(), cos(), tan()
<li>Pangkat 2, dan faktorial
<li>Tombol Fungsi Memori :  MC, MR, M+, M-

<h2>Fitur pada Kalkulator</h2>
<li>Angka dari 0 hingga 9
<li>Mendukung perhitungan desimal
<li>Fitur riwayat dengan scrollbar untuk melihat perhitungan sebelumnya
<li><b>AC :</b> Tombol All Clear untuk menghapus seluruh tampilan
<li><b>CE :</b> Tombol Clear Entry untuk menghapus satu karakter (backspace)
<li><b> = :</b> Untuk menghitung hasil ekspresi
<li><b> + :</b> Untuk penjumlahan
<li><b> - :</b> Untuk pengurangan
<li><b> * :</b> Untuk perkalian
<li><b> / :</b> Untuk pembagian
<li><b> % :</b> Untuk menghitung sisa hasil bagi
<li><b>x! :</b> Untuk menghitung nilai faktorial
<li><b>&#8731 :</b> Untuk menghitung akar pangkat dua
<li><b>sin(value) :</b> Mengembalikan nilai sinus dari sebuah angka (dalam radian)
<li><b>cos(value) :</b> Mengembalikan nilai cosinus dari sebuah angka (dalam radian)
<li><b>tan(value) :</b> Mengembalikan nilai tangen dari sebuah angka (dalam radian)
<li><b>MC :</b> Menghapus nilai yang tersimpan di memori
<li><b>MR :</b> Menampilkan kembali nilai yang tersimpan di memori
<li><b>M+ :</b> Menambahkan angka ke memori
<li><b>M- :</b> Mengurangi angka dari memori
<li><b>MS :</b> Menyimpan angka ke memori
